# API Testing – Add Place & Delete Place

This project demonstrates API testing for Add Place and Delete Place endpoints using REST APIs.  
The purpose of this project is to validate that a place can be successfully **created and deleted** using API requests.

---

## Project Overview

This repository contains API tests that:

1. **Add a new place** using a POST request.
2. **Validate the response** to ensure the place was created successfully.
3. **Delete the created place** using a DELETE request.
4. **Verify that the place was removed successfully**.

---

## Technologies Used

- REST API
- Java
- JSON
- Postman / REST Assured
- Cucumber(Gherkin)
- TestNG
- GitHub for version control

---

## API Endpoints Tested
1. AddPlaceAPI("/maps/api/place/add/json")
2. GetPlaceAPI("/maps/api/place/get/json")
3. DeletePlaceAPI("/maps/api/place/delete/json")
